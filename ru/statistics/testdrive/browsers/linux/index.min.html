Методика сравнения

Результаты браузеров в категориях: основное, функциональность, производительность рассчитываются как среднее значение по каждому из указанных параметров, умноженное на коэффициент заполненности данных соответствующей категории. Итоговый результат рассчитывается как среднее значение результатов по этим категориям, взятых с коэффициентами соответствующими важности данной категории. Коэффициенты категорий варьируются в зависимости от операционной системы и в среднем приблизительно составляют: основное - 0.3, функциональность - 0.2, производительность - 0.5.
