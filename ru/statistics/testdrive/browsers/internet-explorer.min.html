Internet Explorer — браузер, созданный компанией Microsoft, на данный момент заменённый браузером Microsoft Edge. Браузер работает на движке Trident и доступен во всех версиях Windows, выпущенных за последние годы, однако с выходом Windows 10 более не является браузером по умолчанию, а поставляется лишь для обеспечения совместимости. Также существовала версия браузера для операционной системы macOS. Является одним из наиболее популярных браузеров.
