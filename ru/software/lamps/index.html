Lamps СКАЧАТЬ




Описание

Игроку необходимо перестроить схему, чтобы добавить в неё большее число ламп за ограниченное количество действий. Если при создании цепи в ней больше ламп, чем было в предыдущие разы, то игрок получает дополнительные действия.

Управление
  • Стрелки/мышь - выбор элемента цепи для вращения
  • Пробел/левая кнопка мыши - вращение элемента
