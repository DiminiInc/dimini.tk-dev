Mozilla Firefox Mozilla Foundation https://mozilla.org/firefox

Mozilla Firefox is a free and open-source web browser developed by the Mozilla Foundation and its subsidiary, the Mozilla Corporation. Firefox is available for Windows, OS X and Linux operating systems, with its mobile versions available for Android, and Firefox OS; where all of these versions use the Gecko layout engine to render web pages, which implements current and anticipated web standards, but an additional version – Firefox for iOS was released in late 2015 – that doesn't use Gecko but is forced to use the WebKit-based layout engine built into iOS.
